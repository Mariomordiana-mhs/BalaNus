# BalaNus

## What is BalaNus?

sebuah sistem manajemen perpustakaan berbasis web. BalaNus terinspirasi dari singkatan Bala yang artinya Baca Ala dan Nus yang diambil dari kata Udinus. BalaNus menyediakan aneka buku berkualitas dan terlengkap. BalaNus menyediakan berbagai jenis buku yang menarik mulai dari pendidikan, pengetahuan, novel dan humor.
